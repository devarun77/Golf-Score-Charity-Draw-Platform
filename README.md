# 🏌️‍♂️ Golf Score & Charity Draw Platform

A full-stack web application where users track golf scores, participate in monthly draws, and contribute to charitable causes.

---

## 🚀 Features

### 👤 User Features
- Signup & Login (JWT Authentication)
- Add and view last 5 golf scores
- Participate in monthly draw system
- Select charity with contribution percentage (min 10%)
- Modern dashboard UI

### 🎲 Draw System
- Monthly random number draw (1–45)
- Winners selected based on matching scores
- Multiple winners supported

### ❤️ Charity System
- Choose preferred charity
- Adjustable contribution percentage
- Integrated into user profile

### 👑 Admin Panel
- View all users
- Run monthly draw
- View winners
- Manage subscriptions
- Edit user scores
